1 How to configure Eclipse EGIT to work with gh-pages of github?

While trying to create this github page for #howifixedit project, i couldn't find a direct article on how-tos.
The github help page provided a article on configuring it manually in command line.
Link: https://help.github.com/articles/creating-project-pages-manually/

After reading through many help pages and websites, i found creating new branch (in the name gh-pages) from Eclipse itself.
Following lists the steps:
  • Open Git Repositories perspective in Eclipse. Note this perspective is available when we install EGit plugin in Eclipse.
  • Right click the project repository for which we need to create github project page
  • Choose Switch To option, in that choose New Branch
  • Enter the branch name as gh-pages, please note the Source should be master
  • Click the checkbox Configure upstream for push and pull
  • Choose Merge upstream commits into local branch
  • Right click repository again and click on Commit
  • Once commit is over you can go to http://.github.io/
  • Any change in master branch needs to be done in gh-pages. This may not be correct, need to find if any better way is there
2 Hitting java.lang.OutOfMemoryError: Java heap space in tomcat7 in Digital Ocean server with 512M RAM

Recently while helping my friend to test his java application in my DigitalOcean server, even after restarting the tomcat7 following error is thrown while accessing the page:
The proxy could not connect to the destination in time. Please verify the site you are attempting to access and retry.
URL: http://xyz:8080/project

My DigitalOcean droplet is Ubuntu 12.04 with LAMP configured. Tomcat was installed separately.
Though restarting tomcat7 was successful, in /var/lib/tomcat7/logs/catalina.2015-06-23.log following error is seen:
java.lang.OutOfMemoryError: Java heap space

Most of the solutions online suggested, setting JAVA_OPTS with increased -Xmx value. But, none of the solutions were specific to the configuration in DigitalOcean (Ubuntu 12.04).

Finally one of the comments to this post on DigitalOcean's blog suggested that, we need to update the JAVA_OPTS in /etc/default/tomcat7.

In /etc/default/tomcat7 the default -Xmx value was set to 128M, after increasing that value to 256M and restarting the tomcat7 resolved the problem.
3 Making SimpleHTTPServer in python to support different MIME types

When i came to know about SimpleHTTPServer of python, i got hooked to it. To temporarily host some of my folders i used it extensively. But, one thing which needed improvement was to make the SimpleHTTPServer to support different MIME types.

 Usage & example:
Format:
cd /home/somedir
python -m SimpleHTTPServer

Example:
cd /home/prasbala/codeSamples
python -m SimpleHTTPServer 8089

If the directory has a file named index.html, that file will be served as the initial file. If there is no index.html, then the files in the directory will be listed.

To access the html/directory following URL can be entered in browser:
http://:
http://slc03qhm.us.oracle.com:8089/

Supporting more mimetypes: By default the SimpleHTTPServer supports limited number of mimetypes (like .c, .h, .py). To support more mimetypes following part of SimpleHTTPServer needs to be changed:
extensions_map.update({
				        '': 'application/octet-stream', # Default
				        '.py': 'text/plain',
				        '.c': 'text/plain',
				        '.h': 'text/plain',
				        })

To the following with more mimetypes:
				    extensions_map.update({
				        '': 'application/octet-stream', # Default
				        '.py': 'text/plain',
				        '.c': 'text/plain',
				        '.h': 'text/plain',
				        '.java': 'text/plain',
				        '.tsc': 'text/plain',
				        '.sql': 'text/plain',
				        '.log': 'text/plain',
				        '.out': 'text/plain',
				        '.sav': 'text/plain',
				        '.txt': 'text/plain',
				        '.sh': 'text/plain',
				        '.tcov': 'text/plain',
				        })

To find the SimpleHTTPServer location run the following command:
locate SimpleHTTPServer

To edit the file, edit it as sudo user:
sudo vim /usr/lib64/python2.4/SimpleHTTPServer.py
4 How to remove the lines which match & doesn't match the pattern using VIM? In vim we can delete lines matching search pattern using following regex:
:g/<PATTERN>/d

Example:
:g/passwd/d

In order to delete the lines which doesn't match the search criteria following regex can be used:
:g!/<PATTERN>/d
:v/<PATTERN>/d

Example:
:g!/negative/d
:v/negative/d
5 Linux command to know which process is using a particular port Format:
lsof -i:<PORT_NUMBER>

Example:
lsof -i:8080

Some processes/pids are only available to root. For example to see the list of processes listening on some primary ports(25, 22, etc) we need to run the command as superuser.

Example:
sudo lsof -i :22
sudo lsof -i :25
